Lesson: SQL-095 — CROSS JOIN
================================

Practice Questions

Question 1:
Write a CROSS JOIN between products and stores. How many rows result from 3 products × 3 stores? Display all combinations.

Question 2:
Write a CROSS JOIN to generate a product-store availability matrix. Display store_name, product_name, and price for all combinations.

Question 3:
The inventory team needs to plan which products should be stocked in which stores. Write a CROSS JOIN between products and stores to generate all possible product-store pairings. Display store_name, product_name, price, and region.

Question 4:
Write a CROSS JOIN between stores and a manually created list: Write CROSS JOIN with a derived table: SELECT s.store_name, p.product_name FROM stores s CROSS JOIN products p WHERE p.category = 'Electronics'. Show only Electronics in all stores.
