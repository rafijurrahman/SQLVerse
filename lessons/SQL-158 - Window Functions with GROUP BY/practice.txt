Lesson: SQL-158 — Window Functions with GROUP BY
================================

Practice Questions

Question 1:
Write a query combining GROUP BY and a window function:
SELECT category, SUM(revenue) AS cat_total, SUM(SUM(revenue)) OVER () AS grand_total
FROM sales GROUP BY category.
(Window functions operate on the GROUP BY result set.)

Question 2:
Write a query with GROUP BY region, then RANK the regions by their total revenue:
WITH reg_totals AS (SELECT region, SUM(revenue) AS total FROM sales GROUP BY region)
SELECT region, total, RANK() OVER (ORDER BY total DESC) AS rank FROM reg_totals.

Question 3:
Monthly report: GROUP BY month, calculate monthly total, then use a window function to add running monthly cumulative:
WITH monthly AS (SELECT MONTH(sale_date) AS month, SUM(revenue) AS monthly_rev FROM sales GROUP BY MONTH(sale_date))
SELECT month, monthly_rev, SUM(monthly_rev) OVER (ORDER BY month) AS cumulative FROM monthly.

Question 4:
Write a query that combines GROUP BY and multiple window functions:
GROUP BY region and category, then use RANK() within each region to rank categories by revenue.
Display: region, category, total_revenue, and RANK() within region.
