Lesson: SQL-146 — FIRST_VALUE()
================================

Practice Questions

Question 1:
Write a query using FIRST_VALUE() to show the first (lowest date) sale's revenue alongside each row:
FIRST_VALUE(revenue) OVER (ORDER BY sale_date) AS first_sale_revenue.

Question 2:
Write a query to show the first sale within each region:
FIRST_VALUE(revenue) OVER (PARTITION BY region ORDER BY sale_date) AS first_regional_revenue.

Question 3:
The analytics team wants to compare each sale to the first sale of the year as a baseline.
Display: sale_date, product, revenue, FIRST_VALUE(revenue) OVER (ORDER BY sale_date) as "Baseline Revenue",
and revenue - FIRST_VALUE(revenue) OVER (ORDER BY sale_date) as "Change from First".

Question 4:
Write a query to find the first (chronologically earliest) product sold in each category:
FIRST_VALUE(product) OVER (PARTITION BY category ORDER BY sale_date) AS first_product.
Display category, sale_date, product, and first_product.
