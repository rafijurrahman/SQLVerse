Lesson: SQL-054 — DATE_FORMAT()
================================

Practice Questions

Question 1:
Write a query to format appointment_date in DD-MM-YYYY format: DATE_FORMAT(appointment_date, '%d-%m-%Y'). Label it "Formatted Date".

Question 2:
Write a query to format appointment_date as the full month name and year: DATE_FORMAT(appointment_date, '%M %Y') as "Month Year".

Question 3:
The hospital appointment card needs dates in readable format. Write a query to display patient_name, doctor_name, and DATE_FORMAT(appointment_date, '%d %M %Y') as "Appointment Date" and TIME_FORMAT(appointment_time, '%h:%i %p') as "Appointment Time".

Question 4:
Write a query to display appointment_id, patient_name, and DATE_FORMAT(appointment_date, '%W, %d %M %Y') as "Full Date" which includes the day of the week. Sort by appointment_date.
