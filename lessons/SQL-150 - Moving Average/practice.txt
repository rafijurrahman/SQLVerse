Lesson: SQL-150 — Moving Average
================================

Practice Questions

Question 1:
Write a 3-period moving average using ROWS BETWEEN:
AVG(revenue) OVER (ORDER BY sale_date ROWS BETWEEN 2 PRECEDING AND CURRENT ROW) AS moving_avg_3.

Question 2:
Write a 5-period moving average:
AVG(revenue) OVER (ORDER BY sale_date ROWS BETWEEN 4 PRECEDING AND CURRENT ROW) AS moving_avg_5.

Question 3:
Analysts use a 3-day moving average to smooth sales trends. Write a query to display:
sale_date, revenue, AVG(revenue) OVER (ORDER BY sale_date ROWS BETWEEN 2 PRECEDING AND CURRENT ROW) as "3-Day Moving Avg".
Round to 2 decimal places.

Question 4:
Write a query showing both 3-period and 7-period moving averages side by side (15 sales = not enough for 7, use whatever rows available):
AVG(revenue) OVER (ORDER BY sale_date ROWS BETWEEN 2 PRECEDING AND CURRENT ROW) AS ma3,
AVG(revenue) OVER (ORDER BY sale_date ROWS BETWEEN 6 PRECEDING AND CURRENT ROW) AS ma7.
