Lesson: SQL-049 — MINUTE()
================================

Practice Questions

Question 1:
Write a query to extract the minute from appointment_time: MINUTE(appointment_time). Label it "Appointment Minute".

Question 2:
Write a query to find all appointments that start exactly on the hour (MINUTE(appointment_time) = 0).

Question 3:
The clinic books appointments at 00 or 30 minutes. Write a query to find any appointments that were not booked at the standard times: MINUTE(appointment_time) NOT IN (0, 15, 30, 45).

Question 4:
Write a query to display appointment_id, patient_name, HOUR(appointment_time) as "Hour", MINUTE(appointment_time) as "Minute", and SECOND(appointment_time) as "Second". Sort by hour, then minute.
