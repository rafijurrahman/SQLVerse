Lesson: SQL-153 — Cumulative Sum
================================

Practice Questions

Question 1:
Write a query for cumulative sum ordered by sale_date:
SUM(revenue) OVER (ORDER BY sale_date) AS cumulative_sum.

Question 2:
Write a cumulative sum per category:
SUM(revenue) OVER (PARTITION BY category ORDER BY sale_date) AS category_cumulative.

Question 3:
The financial dashboard tracks when cumulative revenue crosses key milestones (5000, 10000, 20000).
Write: SUM(revenue) OVER (ORDER BY sale_date) AS cumulative, and flag each milestone crossed.
Use CASE: WHEN cumulative >= 20000 THEN '20K+' WHEN cumulative >= 10000 THEN '10K+' WHEN cumulative >= 5000 THEN '5K+' ELSE 'Under 5K'.

Question 4:
Write cumulative sum of both revenue AND cost, then calculate cumulative profit:
SUM(revenue) OVER (ORDER BY sale_date) - SUM(cost) OVER (ORDER BY sale_date) AS cumulative_profit.
Display date, revenue, cost, cumulative_profit.
