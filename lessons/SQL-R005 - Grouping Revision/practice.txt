Lesson: SQL-R005 — Grouping Revision
================================

Practice Questions

Question 1:
Write a query to display: category, COUNT(*) as "Items", SUM(quantity) as "Units", SUM(quantity * unit_price) as "Revenue". Group by category. Order by revenue descending.

Question 2:
Write a query to display per store: total revenue, total cost, profit (revenue - cost), and profit margin percentage. Sort by profit descending.

Question 3:
Management wants the top-2 revenue-generating stores. Write a query grouped by store, ordered by total revenue descending, limited to 2.

Question 4:
Write a full report query:
- FROM order_items
- WHERE unit_price > 100
- GROUP BY store, category
- HAVING SUM(quantity * unit_price) > 1500
- SELECT store, category, SUM revenue, AVG unit_price
- ORDER BY revenue DESC
