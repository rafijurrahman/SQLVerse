Lesson: SQL-099 — INNER JOIN with GROUP BY
================================

Practice Questions

Question 1:
Write a LEFT JOIN between customers and orders, then COUNT(*) orders per customer using GROUP BY customer_id.

Question 2:
Write a query to show each customer's name and their total order count. Include customers with 0 orders using LEFT JOIN and COALESCE(COUNT(order_id), 0).

Question 3:
The CRM team wants customer engagement metrics. Show each customer's name, city, total orders placed, and total amount spent. Sort by total orders descending.

Question 4:
Write a LEFT JOIN with COUNT and HAVING: find customers with more than 1 order. Display customer_id, name, and order count. Sort descending.
